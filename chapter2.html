Headings

HTML provides six levels of headings, from <h1> to <h6>. Each level represents a different degree of importance:

                
<h1>This is a heading level 1</h1>
<h2>This is a heading level 2</h2>
<h3>This is a heading level 3</h3>
<h4>This is a heading level 4</h4>
<h5>This is a heading level 5</h5>
<h6>This is a heading level 6</h6>

Result:

This is a heading level 1

This is a heading level 2

This is a heading level 3

This is a heading level 4

This is a heading level 5
This is a heading level 6

Links

Links are created using the <a> tag. A hyperlink can be inserted into an HTML5 document using the following syntax:

                
<a href="http://www.example.com">Click here</a>

The href attribute specifies the URL of the page the link goes to.

Result:

Click here to visit example.com

Images

HTML5 allows you to embed images using the <img> tag. Images are used to enhance the visual appearance of a webpage. Here's how to include an image:

                
<img src="path/to/image.jpg" alt="Description of image">

Specialized Characters and Horizontal Rules

HTML provides ways to include special characters and horizontal rules:

                
<p>This is an example of using &copy; to display © symbol.</p>
<hr>

Result:

This is an example of using © to display © symbol.

Lists

HTML5 supports both ordered and unordered lists:

                
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol>
    <li>First</li>
    <li>Second</li>
    <li>Third</li>
</ol>

Result:

  • Item 1
  • Item 2
  • Item 3
  1. First
  2. Second
  3. Third

Tables

HTML5 allows you to create tables to display data in a structured format:

                
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Result:

Header 1 Header 2
Data 1 Data 2

Basic Forms

HTML5 allows the creation of forms to collect user input:

                
<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

Internal Linking

Internal links allow navigation within the same webpage:

                
<a href="#top">Back to Top</a>

Result:

Back to Top

Exercises

1. Create a simple HTML5 page with headings, links, and images.

Solution 
                        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Simple Page</title>
</head>
<body>
    <h1>Welcome</h1>
    <p>Visit <a href="http://www.example.com">Example</a>.</p>
    <img src="path/to/image.jpg" alt="Example Image">
</body>
</html>
