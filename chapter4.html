Difference between Inline, Embedded, and External Style Sheets

CSS can be applied to HTML documents in three ways:

  • Inline Style Sheet: Uses the style attribute directly in HTML elements.
  • Embedded Style Sheet: Uses the <style> tag within the <head> section of the HTML document.
  • External Style Sheet: Uses the <link> tag to link an external CSS file.

Inline Style Example:

                
<p style="color: red;">This is an inline styled paragraph.</p>

This is an inline styled paragraph.

Embedded Style Example:

                
<style>
    .embedded-style {
        color: blue;
    }
</style>
<p class="embedded-style">This is an embedded styled paragraph.</p>

This is an embedded styled paragraph.

External Style Example:

                
<link rel="stylesheet" href="styles.css">
<p class="external-style">This is an external styled paragraph.</p>

This is an external styled paragraph.

CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. Below are some basic CSS attributes:

Colors in CSS can be specified in various ways, such as using color names, HEX values, RGB values, etc.:

                
/* Color names */
color: red;

/* HEX values */
color: #ff0000;

/* RGB values */
color: rgb(255, 0, 0);

Positioning Elements

CSS positioning properties allow you to position elements on a web page in different ways. The two main types of positioning are absolute and relative:

Absolute Positioning

                
.absolute {
    position: absolute;
    top: 50px;
    left: 100px;
}

Result:

This is an absolutely positioned element.

Relative Positioning

                
.relative {
    position: relative;
    top: 20px;
    left: 30px;
}

Result:

This is a relatively positioned element.

Backgrounds

CSS provides various properties to style the background of an element:

                
background-color: lightblue;
background-image: url('images/example2.png');
background-repeat: no-repeat;
background-size: cover;

Result:

This div has a background image and color.

Element Dimensions

You can set the width and height of an element using the width and height properties:

                
width: 200px;
height: 100px;

Result:

This div has a width of 200px and a height of 100px.

Box Model and Text Flow

The CSS box model is used to define the layout and design of elements. It consists of margins, borders, padding, and the actual content:

                
div {
    width: 100px;
    height: 100px;
    margin: 10px;
    padding: 20px;
    border: 5px solid black;
}

Result:

This div demonstrates the box model.

Text flow in CSS can be controlled using properties like float and clear:

                
.float-left {
    float: left;
}
.clear-both {
    clear: both;
}

Result:

This div floats to the left.
This text flows around the floating div.
This text is below the floating div.

Exercises

1. Create an HTML page with an inline style, embedded style, and external style. Show the differences.

Solution 
                        
<!-- Inline Style -->
<p style="color: red;">This is an inline styled paragraph.</p>

<!-- Embedded Style -->
<style>
    .embedded-style {
        color: blue;
    }
</style>
<p class="embedded-style">This is an embedded styled paragraph.</p>

<!-- External Style -->
<link rel="stylesheet" href="styles.css">

2. Create an HTML page with a div that uses absolute and relative positioning. Demonstrate the differences.

Solution 
                        
<!-- Absolute Positioning -->
<div style="position: absolute; top: 50px; left: 100px; background-color: lightgray;">This is an absolutely positioned element.</div>

<!-- Relative Positioning -->
<div style="position: relative; top: 20px; left: 30px; background-color: lightgray;">This is a relatively positioned element.</div>
