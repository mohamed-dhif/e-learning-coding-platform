Introduction to Functions

Functions are blocks of code designed to perform a specific task. A function is executed when something invokes it (calls it).

                
<script>
    function myFunction() {
        console.log("Hello, world!");
    }
    myFunction();
</script>

Creating Functions

Functions are defined using the function keyword, followed by a name, parentheses, and a block of code enclosed in curly braces.

                
<script>
    function add(a, b) {
        return a + b;
    }
    console.log(add(5, 3)); // Outputs: 8
</script>

Scopes

JavaScript has function scope and block scope. Variables defined inside a function are not accessible from outside the function.

                
<script>
    function scopeExample() {
        var localVar = "I'm local";
        console.log(localVar);
    }
    scopeExample();
    console.log(localVar); // Error: localVar is not defined
</script>

Recursion

Recursion is a technique where a function calls itself. This is useful for tasks that can be broken down into similar subtasks.

                
<script>
    function factorial(n) {
        if (n === 0) {
            return 1;
        }
        return n * factorial(n - 1);
    }
    console.log(factorial(5)); // Outputs: 120
</script>

Random Dice Roll Example

This example demonstrates a function that simulates rolling a six-sided dice.

                
<script>
    function rollDice() {
        return Math.floor(Math.random() * 6) + 1;
    }
    console.log("You rolled a " + rollDice());
</script>

Solution

<head>
    <meta charset="UTF-8">

</head>
