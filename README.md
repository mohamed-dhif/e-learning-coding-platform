# E-Learning Coding Platform

## Overview

This project is a simple web-based e-learning platform developed as part of the **Web Design** course at university.

The platform provides coding-related learning content through multiple chapters and includes interactive quizzes and an HTML compiler interface to enhance the learning experience.

This project was developed collaboratively by a team of two students and is presented here as part of my academic portfolio.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Features

- Multi-page website
- Coding learning chapters
- Interactive quizzes
- HTML compiler interface
- Responsive and user-friendly interface
