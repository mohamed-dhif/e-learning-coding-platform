Introduction to Control Statements

Control statements in JavaScript allow you to control the flow of execution of your code based on conditions and loops.

The if Statement

The if statement is used to execute a block of code only if a specified condition is true.

                
<script>
    var x = 10;
    if (x > 5) {
        console.log("x is greater than 5");
    }
</script>

The if...else Statement

The if...else statement executes one block of code if a condition is true and another block of code if the condition is false.

                
<script>
    var x = 10;
    if (x > 5) {
        console.log("x is greater than 5");
    } else {
        console.log("x is not greater than 5");
    }
</script>

The while Loop

The while loop repeats a block of code as long as a specified condition is true.

                
<script>
    var i = 0;
    while (i < 5) {
        console.log("Iteration " + i);
        i++;
    }
</script>

2. Create an HTML page with a JavaScript while loop that prints numbers from 1 to 5.

Solution 
                        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>JavaScript while Loop</title>
</head>
<body>
    <script>
        var i = 1;
        while (i <= 5) {
            console.log(i);
            i++;
        }
    </script>
</body>
</html>
