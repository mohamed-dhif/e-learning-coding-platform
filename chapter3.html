Autocomplete Attribute

The autocomplete attribute specifies whether a form or input field should have autocomplete enabled:

                
<form autocomplete="on">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

Result:

Datalist Element

The <datalist> element specifies a list of pre-defined options for an <input> element:

                
<label for="browser">Choose your browser from the list:</label>
<input list="browsers" id="browser" name="browser">
<datalist id="browsers">
    <option value="Chrome">
    <option value="Firefox">
    <option value="Safari">
    <option value="Opera">
    <option value="Edge">
</datalist>

Result:

Page Structures

HTML5 introduces nine new page structure elements to organize content:

  • <header>
  • <nav>
  • <article>
  • <section>
  • <aside>
  • <footer>
  • <figure>
  • <figcaption>
  • <main>

These elements help create more semantic HTML documents.

Collapsible Sections with Summary and Details

The <summary> and <details> elements create collapsible sections:

                
<details>
    <summary>More Information</summary>
    <p>This is additional information that can be collapsed.</p>
</details>

Exercises

1. Create an HTML page with various form input types.

Solution 
                        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Form Input Types</title>
</head>
<body>
    <form>
        <label for="textInput">Text Input:</label>
        <input type="text" id="textInput" name="textInput">
        <br>
        <label for="emailInput">Email Input:</label>
        <input type="email" id="emailInput" name="emailInput">
        <br>
        <label for="dateInput">Date Input:</label>
        <input type="date" id="dateInput" name="dateInput">
        <br>
        <label for="colorInput">Color Input:</label>
        <input type="color" id="colorInput" name="colorInput">
        <br>
        <label for="rangeInput">Range Input:</label>
        <input type="range" id="rangeInput" name="rangeInput" min="0" max="100">
        <br>
        <label for="numberInput">Number Input:</label>
        <input type="number" id="numberInput" name="numberInput" min="0" max="100">
        <br>
        <input type="submit" value="Submit">
    </form>
</body>
</html>
