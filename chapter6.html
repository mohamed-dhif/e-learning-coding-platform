Introduction to JavaScript

JavaScript is a versatile programming language used to create dynamic and interactive web content. It is a client-side scripting language that can be embedded directly into HTML pages.

                
<script>
    alert('Hello, world!');
</script>

Including JavaScript in an HTML file can be done using the <script> tag. The <script> tag can be placed in the <head> or <body> section of the HTML document:

                
<head>
    <script>
        alert('Hello from the head section!');
    </script>
</head>

<body>
    <script>
        alert('Hello from the body section!');
    </script>
</body>

Variables

Variables in JavaScript are used to store data values. You can declare a variable using the var, let, or const keyword.

                
<script>
    var x = 5;
    let y = 10;
    const z = 15;
    console.log(x, y, z);
</script>

Simple Calculations

JavaScript can be used to perform simple calculations such as addition, subtraction, multiplication, and division:

                
<script>
    var a = 10;
    var b = 20;
    var sum = a + b;
    var difference = b - a;
    var product = a * b;
    var quotient = b / a;
    console.log('Sum: ' + sum);
    console.log('Difference: ' + difference);
    console.log('Product: ' + product);
    console.log('Quotient: ' + quotient);
</script>

Using document.write()

The document.write() method can be used to write HTML content directly into the document. This method can also include JavaScript variables:

                
<script>
    var name = "John";
    document.write("<p>Hello, " + name + "! Welcome to JavaScript learning.</p>");
</script>

Exercises

1. Create an HTML page with a JavaScript function that displays an alert message when a button is clicked.

Solution 
                        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>JavaScript Alert</title>
</head>
<body>
    <script>
        function showAlert() {
            alert('Hello, world!');
        }
    </script>
    <button onclick="showAlert()">Click me</button>
</body>
</html>

2. Create an HTML page with an input field and a button. When the button is clicked, display the entered text in an alert box.

Solution 
                        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>JavaScript Input Alert</title>
</head>
<body>
    <script>
        function showInput() {
            var userInput = document.getElementById('userInput').value;
            alert(userInput);
        }
    </script>
    <input type="text" id="userInput" placeholder="Enter something">
    <button onclick="showInput()">Show Input</button>
</body>
</html>
