Text Shadows

Text shadows add depth and emphasis to text:

                
/* CSS */
text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5);

This is an example of text shadow.

Rounded Corners

Rounded corners make elements look softer and more modern:

                
/* CSS */
border-radius: 15px;
This div has rounded corners.

Box Shadows

Box shadows create a sense of depth around elements:

                
/* CSS */
box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);
This div has a box shadow.

Linear Gradients

Linear gradients transition smoothly between colors along a line:

                
/* CSS */
background: linear-gradient(to right, red, yellow);
This div has a linear gradient background.

Radial Gradients

Radial gradients transition smoothly between colors radiating from a central point:

                
/* CSS */
background: radial-gradient(circle, blue, green);
This div has a radial gradient background.

Multiple Background Images

CSS allows you to set multiple background images for an element:

                
/* CSS */
background: url('images/5bg1.jpg'), url('images/5bg2.jpg');
This div has multiple background images.

Image Borders

CSS allows you to set an image as a border for an element:

                
/* CSS */
border-image: url('images/5bg1.jpg') 30 30 round;
This div has an image border.

Simple Animation

CSS allows you to create simple animations using the @keyframes rule:

                
/* CSS */
@keyframes example {
    0% {background-color: red;}
    50% {background-color: yellow;}
    100% {background-color: red;}
}
.simple-animation {
    animation: example 5s infinite;
}
This div has a simple animation.

Simple Transition

CSS allows you to create simple transitions for changes in CSS properties:

                
        /* CSS */
        .simple-transition {
            width: 100px;
            height: 100px;
            background-color: blue;
            transition: width 2s, height 2s, background-color 2s;
        }
        .simple-transition:hover {
            width: 200px;
            height: 200px;
            background-color: green;
        }
Hover over this div to see the transition effect.

Exercises

1. Create a simple HTML page demonstrating text shadows and box shadows.

Solution 
                        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Text and Box Shadows</title>
    <style>
        .text-shadow {
            text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5);
        }
        .box-shadow {
            box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);
        }
    </style>
</head>
<body>
    <p class="text-shadow">This text has a shadow.</p>
    <div class="box-shadow">This box has a shadow.</div>
</body>
</html>

2. Create an HTML page demonstrating simple animations and transitions.

Solution 
                        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Animations and Transitions</title>
    <style>
        @keyframes example {
            0% {background-color: red;}
            50% {background-color: yellow;}
            100% {background-color: red;}
        }
        .simple-animation {
            animation: example 5s infinite;
        }
        .transition {
            width: 100px;
            height: 100px;
            background-color: blue;
            transition: width 2s, height 2s, background-color 2s;
        }
        .transition:hover {
            width: 200px;
            height: 200px;
            background-color: green;
        }
    </style>
</head>
<body>
    <div class="simple-animation">This box has an animation.</div>
    <div class="transition">Hover over this box to see the transition.</div>
</body>
</html>
